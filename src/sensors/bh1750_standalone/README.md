# BH1750FVI 光照传感器测试程序

基于 ROHM BH1750FVI 数字环境光传感器，通过 I2C 读取光照强度。

---

| 文件 | 说明 |
|------|------|
| `bh1750_test.c` | C 源代码 |
| `bh1750_test` | 交叉编译的 ARM 可执行文件 |
| `Makefile` | 编译脚本 |

## 🔌 硬件连接

| BH1750 | P9 引脚 | 说明 |
|--------|---------|------|
| VCC | Pin 1 (VCC_3V3) | 3.3V 供电（不是 5V！） |
| GND | Pin 6 (GND) | 地 |
| SDA | Pin 3 (I2C2_SDA) | I2C 数据 |
| SCL | Pin 5 (I2C2_SCL) | I2C 时钟 |

- I2C 总线: `/dev/i2c-2`
- I2C 地址: `0x23`（ADDR 引脚接 GND）
- 注意: BH1750 是 **3.3V** 器件，不能接 5V

## 🚀 使用方法

### 单次读取
```bash
./bh1750_test -1 /dev/i2c-2
```

### 连续读取（默认，Ctrl+C 停止）
```bash
./bh1750_test /dev/i2c-2
```

### 每 2 秒读取一次
```bash
./bh1750_test -i 2000 /dev/i2c-2
```

### 超高精度单次读取 (0.5 lx)
```bash
./bh1750_test -m once-h2 /dev/i2c-2
```

### 低分辨率快速读取 (4 lx, 16ms)
```bash
./bh1750_test -m cont-l /dev/i2c-2
```

### 详细模式（调试用）
```bash
./bh1750_test -v -1 /dev/i2c-2
```

## 📋 命令参数

| 参数 | 说明 |
|------|------|
| `-1` | 单次高精度读取 |
| `-m <mode>` | 测量模式（见下表） |
| `-i <ms>` | 连续读取间隔（毫秒, 默认 500） |
| `-n <count>` | 单次模式读取次数 |
| `-a <addr>` | I2C 地址（默认 0x23） |
| `-v` | 详细模式 |

### 测量模式

| 模式名 | 命令码 | 分辨率 | 测量时间 | 说明 |
|--------|--------|--------|----------|------|
| `cont-h` | 0x10 | 1 lx | 120ms | 连续高分辨率（默认） |
| `cont-h2` | 0x11 | 0.5 lx | 120ms | 连续超高分辨率 |
| `cont-l` | 0x13 | 4 lx | 16ms | 连续低分辨率（快速） |
| `once-h` | 0x20 | 1 lx | 120ms | 单次高分辨率 |
| `once-h2` | 0x21 | 0.5 lx | 120ms | 单次超高分辨率 |
| `once-l` | 0x23 | 4 lx | 16ms | 单次低分辨率 |

## 📐 计算公式

```
raw = (MSB << 8) | LSB
lux = raw / 1.2
```

## 🐛 故障排除

### 无法打开 I2C 设备
```bash
# 检查 I2C 总线是否存在
ls -l /dev/i2c*

# 扫描 I2C 设备地址
i2cdetect -y -r 2
```

### 读到 0 或负值
- 检查 ADDR 引脚是否接地（地址 0x23）
- 确认供电是 3.3V（不是 5V）
- 上电后需等 120ms 首次测量完成

### 读数不准确
- BH1750 光敏窗口不要被遮挡
- 避免红外光源直射（传感器对红外敏感）
